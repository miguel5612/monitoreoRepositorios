🚀 PASOS PARA EMPEZAR (5 minutos):

┌────────────────────────────────────────────────────────────────┐
│ PASO 1: Abrir Terminal                                        │
└────────────────────────────────────────────────────────────────┘

   1. Abre la carpeta en el Explorador de Archivos
   2. Haz clic derecho en un espacio vacío
   3. Selecciona "Abrir en Terminal"

┌────────────────────────────────────────────────────────────────┐
│ PASO 2: Instalar (solo la primera vez)                       │
└────────────────────────────────────────────────────────────────┘

   Ejecuta en la terminal:
   
   setup.bat
   
   Esto instalará todas las dependencias automáticamente.
   
   NOTA: Necesitas tener instalado:
   • Node.js (https://nodejs.org)
   • Git (https://git-scm.com)

┌────────────────────────────────────────────────────────────────┐
│ PASO 3: Configurar tus Repositorios                          │
└────────────────────────────────────────────────────────────────┘

   1. Abre el archivo: config.js
   2. Modifica la sección "repositories"
   3. Cambia las rutas por las de tus proyectos
   
   EJEMPLO:
   
   {
     id: 'mi-proyecto',
     name: 'Mi Proyecto Backend',
     localPath: 'C:\\Users\\Usuario\\proyectos\\mi-proyecto',
     branches: ['main', 'develop'],
     responsible: 'Tu Nombre',
     team: 'Backend',
     priority: 'high'
   }
   
   IMPORTANTE: Usa doble barra \\ en las rutas

┌────────────────────────────────────────────────────────────────┐
│ PASO 4: Probar                                               │
└────────────────────────────────────────────────────────────────┘

   Ejecuta en la terminal:
   
   npm test
   
   Esto generará un informe inmediatamente.

┌────────────────────────────────────────────────────────────────┐
│ PASO 5: Ver el Resultado                                     │
└────────────────────────────────────────────────────────────────┘

   Abre la carpeta: reports\
   
   Encontrarás:
   • latest.jpg  ← Imagen del informe
   • latest.html ← Versión interactiva
   • latest.json ← Datos en JSON

═══════════════════════════════════════════════════════════════════

📚 COMANDOS ÚTILES:

   npm run demo      Ver informe de ejemplo

   npm start         Modo automático (diario)
